Documento sobre el código

La aplicación esta compuesta por:
  • Archivo html sumatoriacubo.html
  • JavaScript con el código que realiza las operaciones sumatoriacubo.js
Las pruebas estan en:
  • Archivo html testsumatoriacubo.html
  • JavaScript con las pruebas de la aplicación testsumatoriacubo.js
En el archivo sumatoriacubo.js se encuentra el código de la aplicación en el paquete sumatoriacubo. En el paquete se encuentra:
  • La función procesar la cual es ejecutada cuando se hece click en el boton procesar
  • La función stringToEntero que realiza conversión de cadenas a enteros
  • La clase LectorPruebas la cual contiene la lógica que realiza las operaciones requeridas por la prueba
